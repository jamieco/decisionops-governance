# DecisionOps Governance
This repository is the public governance interface for the DecisionOps Platform.

It contains three files, automatically synchronised from the canonical private repository on every merge to main:

- `decisionops_project_instructions.md` — operational rules and governance model
- `decisionops_corpus_catalogue.md` — authoritative inventory of the documentation corpus
- `corpus_manifest.yaml` — machine-readable structural definition of the corpus

These files are published here to support bootstrap verification and governance transparency. All changes are made in the canonical repository only — do not raise PRs or issues here.
